## 1. 서비스 소개

### 1.1 배경 및 목적

인플루언서와 콘텐츠 마케터들은 촬영 현장에서 수많은 '필수 컷' 가이드라인을 확인하며 촬영해야 합니다. QUDO는 촬영 리스트 관리의 번거로움, 구도 설정의 어려움, 모바일에서 찍은 사진을 PC 작업 환경으로 옮기는 비효율을 AI 기술로 혁신하기 위해 탄생했습니다.

### 1.2 핵심 가치

- **누락 없는 완벽함**: 카테고리별 추천 및 AI 캠페인 가이드 분석으로 필수 컷 누락 방지
- **일관된 퀄리티**: 시그니처 구도 오버레이를 통해 브랜드 아이덴티티 유지
- **끊김 없는 흐름**: Web Drop 기술로 모바일 촬영 결과물을 PC 브라우저에서 즉시 활용
- **성장하는 크리에이터**: 뱃지 시스템과 캠페인 플랫폼으로 수익 기회 확장

## 2. 주요 기능

### 2.1 지능형 촬영 계획

- **카테고리별 스마트 컷 추천**: 맛집·제품·여행·상세페이지 카테고리에 맞는 투두 자동 제안
- **AI 캠페인 미션 분석**: 광고 가이드 텍스트를 붙여넣으면 촬영 미션·마감·법적 고지 자동 추출 (연동 예정)

### 2.2 스마트 가이드 카메라

- **AI 촬영 자동 분류 (Shot Verification)**: 촬영 즉시 Gemini 2.5 Flash가 어떤 투두 항목인지 판단하여 자동 체크
- **수동 촬영 모드**: 투두 항목 옆 카메라 버튼으로 해당 항목에 AI 분류 없이 즉시 할당
- **사진 처리 큐**: 연속 촬영 시 API 직렬 처리 + Rate Limit 자동 재시도 + 지수 백오프
- **핀치 줌**: 두 손가락 제스처로 실시간 줌 조절
- **줌 슬라이더 & 프리셋**: 0.5× / 1× / 2× 프리셋 + 연속 슬라이더
- **탭 포커스**: 화면 탭 위치에 포커스 마커 + 노출 슬라이더 연동
- **노출 조절**: 우측 수직 슬라이더로 -1 ~ +1 노출 조절
- **비율 전환**: 4:3 / 16:9 / 1:1 프리뷰 전환
- **삼분할선 가이드**: 설정으로 on/off
- **셀프타이머**: 3초 / 10초 카운트다운 후 자동 촬영
- **시그니처 샷 오버레이**: 나만의 구도를 반투명 레이어로 겹쳐 촬영 (연동 예정)

### 2.3 Web Drop (PC 연동 워크스페이스)

- **보안 링크 발급**: 비밀번호 보호 옵션 포함
- **에셋 그리드**: CSS Grid, 투두 항목별 필터, 드래그 앤 드롭, 클립보드 복사
- **라이트박스**: 전체화면 뷰어, 좌우 화살표, 키보드 탐색, 다운로드

### 2.4 캠페인 찾기

- **탐색**: 배송형(제품 수령 후 리뷰) / 방문형(오프라인 방문 후 콘텐츠)
- **필터 & 검색**: 카테고리 칩 + 실시간 검색
- **신청**: D-Day·경쟁률 확인 후 1탭 신청
- **워크스페이스 자동 생성**: 당선 시 광고주 요구사항이 반영된 워크스페이스 자동 생성

### 2.5 마이페이지 & 뱃지 시스템

- **활동 통계**: 총 워크스페이스·촬영 컷·뱃지 수
- **뱃지 모음**: 10종 (첫 촬영, 완벽 완료, 카테고리 마스터 등)
- **채널 정보 등록**: Instagram / YouTube / TikTok / 블로그
- **플랜 관리**: Free / Pro

| 역할 | 명칭 | 주요 활용 시나리오 |
|------|------|-----------------|

## 5. 서비스 흐름 (User Journey)

1. **시작**: 소셜 로그인(카카오·네이버·Google·Apple) 후 대시보드 진입
2. **캠페인 탐색**: 배송/방문 캠페인 찾기 → 조건 확인 → 신청 (당선 시 워크스페이스 자동 생성)
3. **계획**: 카테고리 선택 또는 캠페인 안내문 붙여넣기로 투두리스트 자동 생성
4. **촬영**: 핀치줌·포커스·노출 등 전문 카메라 기능 + AI 자동 분류로 필수 컷 완성
5. **연동**: PC에서 Web Drop 링크 접속 (비밀번호 보호 옵션)
6. **작업**: 블로그/SNS 에디터로 이미지를 직접 드래그하여 포스팅 완성
7. **성장**: 뱃지 획득으로 활동 기록 및 동기부여
